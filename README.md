## Usage
- Install `pnpm install` [pnpm install doc](https://pnpm.io/installation)
- Run locally `pnpm run dev`

### Local Testing
- uncomment `DUMMY_JSON` and `INITIAL_JSON` from `App.tsx` to see testing data.
